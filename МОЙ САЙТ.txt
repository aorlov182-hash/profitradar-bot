## ✅ **ЧТО ТЕПЕРЬ:**

### **1. Скопируйте ссылку на сайт**
Во вкладке "Deployments" вы видите ссылки вида:
- `https://profitradar-landing.pages.dev` (основная)
- `https://xxxxx.profitradar-landing.pages.dev` (превью для конкретного коммита)

**Скопируйте основную ссылку** и откройте её в браузере — вы увидите ваш лендинг! 🌐

---

### **2. Как обновлять сайт**

Каждый раз, когда вы меняете `landing/index.html`:

```powershell
git add .
git commit -m "Обновил лендинг"
git push
```

Cloudflare **автоматически** пересоберёт и обновит сайт через 1-2 минуты. Ничего больше делать не нужно!

---

### **3. (Опционально) Привяжите свой домен**

Если хотите сайт на своём домене (например, `profitradar.ru`):
1. В Cloudflare Pages перейдите во вкладку **"Custom domains"**
2. Нажмите **"Set up a custom domain"**
3. Введите ваш домен
4. Cloudflare даст DNS-записи для настройки у регистратора домена

---

## 📊 **Где смотреть статистику:**

- **Deployments** — история обновлений
- **Analytics** — статистика посещений
- **Settings** — настройки домена, переменных окружения
