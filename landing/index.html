Частые вопросы

Это бесплатно?
Базовый расчёт бесплатный. Подписка нужна для автообновления данных и алертов.
Какие маркетплейсы поддерживаете?
Сейчас поддерживаем Wildberries и Ozon. В планах — добавить Яндекс.Маркет и другие площадки.
Мои данные в безопасности?
Да! Мы используем только read-only доступ к API маркетплейсов. Это значит, что мы можем только читать данные, но не можем ничего изменить. Ваши API-ключи хранятся в зашифрованном виде.
Сколько времени нужно на подключение?
Подключение API занимает 2-3 минуты. Нужно скопировать API-ключ из личного кабинета селлера и вставить в бота. После этого все расчеты будут автоматическими.
